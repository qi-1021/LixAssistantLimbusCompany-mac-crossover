# LALC 离线安装包说明

## 📦 包含内容

本项目已将 macOS 所需的所有依赖包打包到 `lalc_backend/deps_macos/` 目录中。

- **依赖包数量**: 41 个
- **总大小**: 约 171 MB
- **平台**: macOS (Apple Silicon & Intel)

### 依赖清单

核心依赖包：
- matplotlib (3.10.8) - 数据可视化
- numpy (2.4.3) - 数值计算
- onnxruntime (1.24.3) - AI 推理引擎
- opencv-python-headless (4.13.0) - 图像处理
- pillow (12.1.1) - 图像处理
- plyer (2.1.0) - 系统通知
- psutil (7.2.2) - 系统监控
- pynput (1.8.1) - 输入控制
- pyobjc-framework-Quartz (12.1) - macOS 窗口检测
- pyobjc-framework-Cocoa (12.1) - macOS 系统接口
- pyobjc-core (12.1) - PyObjC 核心
- rapidocr (3.7.0) - OCR 识别
- scipy (1.17.1) - 科学计算
- websockets (16.0) - WebSocket 服务器

以及所有传递依赖（contourpy, cycler, fonttools, shapely 等）。

## 🚀 使用方法

### 方式 1: 离线安装（推荐）

如果你的电脑无法访问 PyPI（网络受限或离线环境）：

```bash
# 1. 解压项目到本地
cd LixAssistantLimbusCompany-master

# 2. 运行离线安装脚本
./安装LALC-mac-离线.sh
```

或使用新版统一脚本（推荐维护者/打包者使用）:

```bash
# 在项目根创建归档（仅当你是打包者时）
./scripts/package_deps.sh

# 给用户分发后，用户运行：
./安装LALC-mac-离线.sh --create-venv --install-deps --install-flutter
```

**说明**：
- 脚本会自动检测本地 `deps_macos/` 目录
- 使用 `pip install --no-index --find-links=deps_macos` 从本地安装
- 如果本地依赖不完整，会自动降级为在线安装

### 方式 2: 在线安装

如果你的电脑可以访问 PyPI：

```bash
./安装LALC-mac.sh
```

## ⚠️ 注意事项

### 1. 平台兼容性

`deps_macos/` 中的依赖包是为 **Apple Silicon (arm64)** 架构编译的。

**Intel Mac 用户**需要重新下载：

```bash
cd lalc_backend
rm -rf deps_macos
mkdir deps_macos

# 为 Intel 下载依赖
pip download --dest deps_macos -r requirements-macos.txt --platform macosx_10_9_x86_64
```

**通用解决方案**（支持 Intel + Apple Silicon）：

```bash
# 下载 universal2 架构的包（如果有）
pip download --dest deps_macos -r requirements-macos.txt --platform macosx_10_13_universal2
```

### 2. Python 版本

依赖包是为 **Python 3.13** 编译的，兼容 Python 3.10-3.13。

如果使用其他 Python 版本，部分包可能需要重新下载：

```bash
# 为特定 Python 版本下载
pip download --dest deps_macos -r requirements-macos.txt --python-version 3.12
```

### 3. Git 克隆问题

如果使用 `git clone` 时遇到大文件问题，可以：

```bash
# 方式 1: 使用 Git LFS（如果仓库配置了 LFS）
git lfs install
git clone https://github.com/HSLix/LixAssistantLimbusCompany.git

# 方式 2: 浅克隆（不包含历史记录）
git clone --depth 1 https://github.com/HSLix/LixAssistantLimbusCompany.git

# 方式 3: 直接下载 ZIP 文件
curl -L https://github.com/HSLix/LixAssistantLimbusCompany/archive/refs/heads/master.zip -o LALC.zip
unzip LALC.zip
```

## 💡 常见问题

**Q: 为什么离线安装失败？**

A: 可能原因：
1. Python 版本不匹配（需要 3.10-3.13）
2. CPU 架构不匹配（Intel vs Apple Silicon）
3. 依赖包损坏

解决方案：
```bash
# 删除旧依赖重新下载
rm -rf lalc_backend/deps_macos
cd lalc_backend
pip download --dest deps_macos -r requirements-macos.txt
```

**Q: 依赖包太大怎么办？**

A: 可以选择性打包：
```bash
# 只打包核心依赖（约 50MB）
pip download --dest deps_macos_lite --no-deps \
    onnxruntime pillow pynput pyobjc-framework-Quartz websockets
```

**Q: 如何更新单个依赖？**

A:
```bash
# 下载最新版本
pip download --dest deps_macos --no-deps "包名"

# 删除旧版本
rm deps_macos/包名-旧版本-*.whl
```
